Biot-Savart Law Interactive Explorer
مستكشف قانون بيو-سافار التفاعلي
الوصف | Description
تطبيق تعليمي تفاعلي شامل لشرح قانون بيو-سافار والمجال المغناطيسي الناشئ عن التيار الكهربائي.يغطي الموصل المستقيم، الملف الدائري، الملف اللولبي، قاعدة اليد اليمنى، واحتواء البلازما.

An interactive educational app covering Biot-Savart Law, magnetic fields from straight wires,circular coils, solenoids, right-hand rule, and plasma confinement.

الميزات | Features
✅ رسوم متحركة تفاعلية لخطوط المجال المغناطيسي
✅ تحكم بتغيير اتجاه التيار ورؤية التأثير بصرياً
✅ قاعدة اليد اليمنى متحركة
✅ اشتقاق رياضي خطوة بخطوة
✅ احتواء البلازما (توكاماك) متحرك
✅ اختيار المادة وتأثير النفاذية المغناطيسية
✅ تقييم تفاعلي نهائي مع أصوات
✅ أمثلة من الحياة اليومية
✅ باللغة العربية مع عناوين إنجليزية
